Lesson 7: The Internet

1. Evolution

The Internet is a global system of interconnected networks using the TCP/IP suite. Its history is marked by rapid advancement:

  • Origins (1960s-1980s): Started with ARPANET, followed by the birth of E-mail and the expansion of institutional networking.
  • Modern Age (1990s-2000s): Tim Berners-Lee invented the World Wide Web (WWW) and HTML; social media platforms like Facebook and Twitter later redefined socialization.
  • Integration (2010s-Present): Shifted toward Cloud Computing, mobile dominance, and a heavy focus on advanced cybersecurity.
2. Connectivity

The Internet has revolutionized daily life by enabling instant global communication and information accessibility.

  • Core Functions: Instant information search, real-time paperless news, and global communication via VOIP and video calls.
  • Internet of Things (IoT): A network of physical objects with sensors that exchange data. While it offers automation, it carries risks of data theft and system corruption.
3. Architecture (Cloud Computing)

Cloud Computing allows for the remote manipulation and configuration of hardware and software resources.

  • IaaS (Infrastructure): Provides virtualized computing resources (e.g., Virtual machines, storage).
  • PaaS (Platform): Provides platforms for developers to build and run applications without worrying about infrastructure.
  • SaaS (Software): Ready-to-use software for end-users accessed via a browser (e.g., Google Drive, Netflix).
  • Deployment Models: Includes Public (shared), Private (dedicated), Community (shared goals), and Hybrid (combined flexibility).
4. Security

Protection of data and transactions is governed by specific protocols and safety behaviors.

  • Protocols: TCP/IP (connection rules), HTTP/HTTPS (transmitting web data), and FTP (file transfers).
  • Best Practices: Utilizing Two-Factor Authorization (2FA), checking for HTTPS (padlock icon), and maintaining "password hygiene" to prevent unauthorized access.
