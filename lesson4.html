Lesson 4: Number System

1. Fundamentals

A number system is a structured method for representing quantities using specific digits and mathematical notation.

  • Primary Role: It provides a bridge between human-readable information and machine-processable data.
  • Function: Makes it easier to compare values and associate quantities with objects for computing efficiency.
2. Bases

There are four primary number systems used in modern technology:

Number System Base Digits Used Common Uses
Decimal 10 0-9 Human calculations, finance
Binary 2 0, 1 Digital electronics, cryptography
Octal 8 0-7 UNIX permissions, UTF-8
Hexadecimal 16 0-9, A-F Color codes, memory addressing
  • Decimal (Base 10): The human standard (0-9); used for high-precision scientific and financial calculations.
  • Binary (Base 2): The foundation of digital electronics (0 and 1); represents True/False states and is vital for cryptography.
  • Octal (Base 8): Uses digits 0–7; primarily used in UNIX file permissions and UTF-8 encoding.
  • Hexadecimal (Base 16): Uses digits 0–9 and letters A–F; a compact system for HTML color codes and memory addressing.
3. Synthesis (To Decimal)

This explains the process of converting any base back into Decimal (Base 10) using positional notation.

Binary to Decimal: Multiply digits by powers of 2 (2^0, 2^1, 2^2, ...)

Octal to Decimal: Multiply digits by powers of 8 (8^0, 8^1, 8^2, ...)

Hexadecimal to Decimal: Convert letters to numbers (A=10 to F=15) then multiply by powers of 16 (16^0, 16^1, 16^2, ...)

4. Decomposition (From Decimal)

Translating Decimal (Base 10) into other bases using two main methods:

  • Division Method: Repeatedly divide the decimal number by the target base (2, 8, or 16) and record the remainders in reverse order.
  • Subtraction Method: (For Binary) Subtract the largest possible power of 2 from the decimal value.
  • Shortcuts: Convert decimal values to binary first, then group them into sets of three (for Octal) or four (for Hexadecimal).

References: COMP001 Module 04: Number System Conversion; Rules of Binary Arithmetic and Signed Numbers.
