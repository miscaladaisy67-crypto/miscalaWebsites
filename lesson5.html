Lesson 5: Digital Logic System

1. Logic

Computing is a formal science built on mathematically definable logic systems. Digital systems function using two distinct binary states:

  • 0 (False/Off): No electrical signal.
  • 1 (True/On): With electrical signal.
  • Machine Decision: By assigning values to these inputs, the system determines "True versus False" outcomes, serving as the basis for all quantitative processing.
2. Gates

The Arithmetic Logic Unit (ALU) uses specialized hardware components called gates to perform operations.

  • AND Gate: Outputs True only if all inputs are True. (Returns 1 if both A and B are 1).
  • OR Gate: Outputs True if at least one input is True. (Returns False only if both are False).
  • NOT Gate: Also known as an Inverter; it flips the input (True becomes False and vice versa). This is vital for implementing 1's complement.
3. Execution

These logical operations are integrated into the CPU's Machine Cycle for high-speed actions.

  • The Execute Phase: During this phase, the CPU performs simple actions to process complex instructions.
  • Hardwired Circuitry: Because these operations are hardwired, they allow the computer to evaluate conditions and route data almost instantaneously.
4. Testing

The logic unit within the ALU is responsible for critical hardware-level tests.

  • Numerical Checks: Determining if a value is negative or comparing two values to see if they are equal.
  • Software Foundation: These tests are the "building blocks" that allow software to use conditional statements (like if-else) to control the flow of a program.
